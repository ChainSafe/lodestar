    Const MAX_RETAINED_ATTESTATIONS_PER_GROUP¶

    • MAX_RETAINED_ATTESTATIONS_PER_GROUP: 4 = 4

    Defined in packages/lodestar/src/chain/opPools/aggregatedAttestationPool.ts:36

    Limit the max attestations with the same AttestationData. Processing cost increases with each new attestation. This number is not backed by data. After merging AggregatedAttestationPool, gather numbers from a real network and investigate how does participation looks like in attestations.

    Const SLOTS_RETAINED¶

    • SLOTS_RETAINED: 3 = 3

    Defined in packages/lodestar/src/chain/opPools/attestationPool.ts:15

    The number of slots that will be stored in the pool.

    For example, if SLOTS_RETAINED == 3 and the pool is pruned at slot 6, then all attestations at slots less than 4 will be dropped and any future attestation with a slot less than 4 will be refused.
