    @chainsafe/lodestar › Globals › sync/range › SyncChain

    Class: SyncChain¶

    Dynamic target sync chain. Peers with multiple targets but with the same syncType are added through the addPeer() hook.

    A chain of blocks that need to be downloaded. Peers who claim to contain the target head root are grouped into the peer pool and queried for batches when downloading the chain.
